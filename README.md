## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Aswinsaipalakonda/landing-page.git

# Navigate into the folder
cd landing-page

# Install dependencies
npm install
```
---

## 🛠️ Running the Project
```bash
# Start the development server
npm run dev
```
